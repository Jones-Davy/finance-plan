# Месячный бюджет

Личный планировщик финансов: траты, диаграммы, недельный лимит и цели.

## Возможности

- Список обязательных и дополнительных трат по категориям
- Диаграмма распределения бюджета (круговая + доход vs траты)
- Недельный и дневной расчёт расходов
- Финансовые цели с прогрессом и рекомендуемой суммой отложений
- Автосохранение в браузере (localStorage)

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно http://localhost:5173).

## Сборка

```bash
npm run build
npm run preview
```
